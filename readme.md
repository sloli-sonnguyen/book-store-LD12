# Bookstore-LD Project

A basic Shopping Application. Client built with React (Web) and React-native (mobile). Server built with Ruby on rails. Use Mysql for database.

## Stack

#### Client (Website)

- Create React App
- React Redux
- Redux Thunk
- React Router
- Styled-components
- Material Ui
- React-elastic-carousel

#### Client (Mobile)

- React Native
- Expo
- React Redux
- Redux Thunk
- React Navigation

#### Server

- Ruby on Rails
- Mysql

## Quick Start

#####The first you need install

- Nodejs
- Npm / Yarn
- Ruby, Rails
- Mysql

```shell
$ git clone https://github.com/sloli-sonnguyen/book-store-LD12
$ cd source
--> You will see 4 folder: backend, database, frontend, mobile
// Step 1: install database
$ cd database --> Next step follow readme file in database folder
// Step 2: install server
$ cd backend --> Next step follow readme file in backend folder
// Step 3:
$ cd frontend
$ npm install
$ npm start
// If you want use mobile app ->  Step 4:
$ cd mobile
$ yarn global add expo-cli
$ yarn install
$ expo start
```
